# miniprojects

A collection of small, self-contained projects spanning web development, data processing, machine learning, and systems programming.

## Projects

### flasktest

A basic Flask web application with a Bootstrap-based frontend. Includes Docker support for containerized deployment.

**Tech:** Python, Flask, Bootstrap, Docker

### minispark

A PySpark script that reads a text file and computes the maximum number of words per line using Spark DataFrames and Spark SQL functions.

**Tech:** Python, PySpark

### pdfparsetest

Extracts structured data from PDF reports (nutritional/chemical analysis) and converts them to CSV. Includes an analysis script that generates visualizations such as bar charts, heatmaps, stacked bar charts, and box plots.

**Tech:** Python, pdfplumber, pandas, matplotlib, seaborn

### tinyChEMBL

A Graph Neural Network trained on the ESOL dataset for molecular solubility prediction. Uses a 2-layer Graph Convolutional Network with global mean pooling.

**Tech:** Python, PyTorch, PyTorch Geometric

### tinyrust

A minimal Rust "Hello, world!" program.

**Tech:** Rust

### tinysite

A personal portfolio/landing page served with Flask and Gunicorn. Dockerized for production deployment.

**Tech:** Python, Flask, Gunicorn, Docker

### weatherapp

A frontend weather application that fetches real-time data from the OpenWeatherMap API. Features animated UI, city search, and displays temperature, conditions, and wind speed.

**Tech:** HTML, CSS, JavaScript, jQuery, OpenWeatherMap API
